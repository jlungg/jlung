# KYC Case Study

Know Your Customer (KYC) case study — CommBank digital identity verification and the Path pattern.

## GitHub Pages

1. Add your remote and push:
   ```bash
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   git push -u origin main
   ```

2. Enable GitHub Pages: **Settings** → **Pages** → **Deploy from branch** → select `main` and `/ (root)`.

3. Your case study will be live at:
   - `https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/` (index)
   - `https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/kyc-path.html`
